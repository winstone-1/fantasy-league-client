# Fantasy League Manager

A sophisticated sports management platform for building, tracking, and optimizing fantasy rosters across multiple sports.

## Features

- **Visual Field Management**: Rendered soccer pitches and basketball courts with dynamic player positioning based on chosen formations.
- **Roster Persistence**: Save and load your custom squads directly to the cloud via a dedicated backend API.
- **Interchangeable Formations**: Switch between various tactical lineups in real-time to optimize team performance.
- **Player Scouting**: Integrated search functionality to find and add new talent to your roster.
- **Responsive UI**: Dark-themed, mobile-friendly interface built with Tailwind CSS for a premium aesthetic.
